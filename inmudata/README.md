# Inmudata Landing Page

Landing page profesional para **Inmudata**, la plataforma SaaS de gestión inmobiliaria.

## 🏠 Sobre Inmudata

Inmudata es un software de gestión integral para propietarios y administradores de fincas que permite:

- **Gestión de Inmuebles** — Fichas completas, contratos, alertas de vencimiento
- **Ingresos & Gastos** — Dashboard financiero, reportes, exportaciones
- **Incidencias** — Tickets con estados, asignación a proveedores, historial

## 🛠️ Stack Técnico

- **HTML5** semántico
- **Tailwind CSS** (CDN)
- **Lucide Icons**
- **AOS** (Animate On Scroll)
- **Google Fonts** (Plus Jakarta Sans + Inter)
- **CSS puro** — Mockups de UI sin imágenes

## ✨ Características

- 🎨 Diseño dark-mode profesional con paleta azul/verde
- 📱 100% responsive (mobile-first)
- 🖥️ Mockups CSS interactivos de la app
- ✨ Animaciones suaves con AOS y CSS
- ♿ Accesibilidad (focus visible, semántica, reduced-motion)
- 🚀 Un único archivo HTML autocontenido
- 🔍 SEO optimizado (meta tags, estructura semántica)
